    10.A website looks great on a laptop but is unreadable on a mobile phone because the user has to zoom in and scroll horizontally. Identify the design principle missing here and explain hoew media queries/fluid grids could fix it.
    Ans:The missing design principle here is Responsive Web Design the website should adapt to different screen sizes on content is readable on all devices.

    How to fix it:
    Media Queries:CSS rules that apply different styles based on the device width(eg. adjusting font size, layout, or hiding /showing elements for mobile screens)
    Fluid Grids: Use percentage based widths instead of fixed pixels so content scales automatically to fit any screen size.

    Result:
    -No horizontal scrolling or zooming.
    -Content is readable and well-organized on mobile,tablet, and desktop.
    -Improved user experience and accessibility.
    Implementing responsive design with media queries and fluid grids ensures the websites works smoothly across all devices.
